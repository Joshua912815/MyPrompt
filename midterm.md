### **第一部分：公司金融导论与财务报表 (Introduction to Corporate Finance & Financial Statements)**

#### 1. 财务管理决策的基本类型与财务经理的角色 (The Basic Types of Financial Management Decisions and the Role of the Financial Manager)

财务经理必须做出三个基本决策：

1.  **资本预算 ($Capital \ Budgeting$ / 投资决策):** 这个问题是“公司应该投资哪些长期项目？” 这需要评估未来现金流的规模、时间和风险，以确定哪些项目能为公司创造价值。这是决定公司未来价值的最重要的决策。
2.  **资本结构 ($Capital \ Structure$ / 融资决策):** 这个问题是“公司应如何为选定的投资项目筹集资金？” 这涉及到确定用于运营融资的最佳债务 ($debt$, 如贷款、债券) 和权益 ($equity$, 如股票) 组合。这种组合被称为资本结构。
3.  **营运资本管理 ($Working \ Capital \ Management$):** 这涉及到日常财务活动的管理。它包括管理公司的短期资产（如存货）和短期负债（如对供应商的付款），以确保企业平稳运行并拥有足够的现金流来支持日常运营。

**财务经理的角色 ($Financial \ Manager's \ Role$)** 是执行这些决策。关键职位包括：
*   **首席财务官 ($Chief \ Financial \ Officer, \ CFO$):** 公司的最高财务负责人，负责战略规划和整体财务管理。
*   **财务主管 ($Treasurer$):** 负责管理公司的现金、信用、资本支出和外部融资活动。
*   **财务总监 ($Controller$):** 负责管理公司的内部财务活动，包括会计、税务和数据处理。

#### 2. 不同商业组织形式（合伙制）的财务影响 (The Financial Implications of the Different Forms of Business Organization (Partnership))

在多种商业组织形式中，**合伙制 ($partnership$)** 具有独特的财务影响：

*   **普通合伙制 ($General \ Partnership$):**
    *   **财务影响:** 所有合伙人分享收益和亏损，并对所有合伙企业的债务承担 **无限责任 ($unlimited \ liability$)**。这意味着债权人不仅可以追索公司投资，还可以追索合伙人的个人资产。这种责任也是“连带”的 ($joint \ and \ several$)，意味着一个合伙人可能要对全部债务负责。
*   **有限合伙制 ($Limited \ Partnership$):**
    *   **财务影响:** 这种形式有一个或多个普通合伙人（负责经营并承担无限责任）和一个或多个有限合伙人。对 **有限合伙人 ($limited \ partners$)** 的关键影响是他们的 **有限责任 ($limited \ liability$)**——他们对企业债务的个人责任仅限于他们在合伙企业中的投资额。他们通常是财务投资者，不参与管理。

#### 3. 财务管理的目标 (The Goal of Financial Management)

财务管理的首要目标是 **最大化公司现有股票的每股当前价值 ($maximize \ the \ current \ value \ per \ share \ of \ the \ existing \ stock$)**，这等同于最大化股东财富 ($shareholder \ wealth$)。

这个目标优于其他潜在目标（如利润最大化或市场份额最大化），因为股价反映了市场对公司未来业绩的综合评估，包括其未来现金流、增长潜力和风险状况。通过最大化股票价值，公司确保正在为其所有者做出最优的投资和融资决策。
maximize value of share / values per share

#### 4. 财务规划的覆盖范围 (The Coverage of Financial Planning)

财务规划为公司的变革和增长制定了指导方针。其关键组成部分是：

*   **规划期 ($Planning \ Horizon$):** 指规划过程关注的长期时间范围，通常为两到五年。
*   **整合 ($Aggregation$):** 将各个项目的资本预算决策合并成一个全面的公司计划。
*   **情景分析 ($Scenario \ Analysis$):** 规划者会创建多种情景——如最差情况、正常情况和最佳情况——以了解潜在结果并为不确定性做准备。
*   **核心要素:** 整个过程以 **销售预测 ($sales \ forecast$)** 为起点，然后用它来创建预测性财务报表，即 **预估报表 ($pro \ forma \ statements$)**。

#### 5. 经理与所有者之间可能出现的利益冲突 (The Conflicts of Interest That Can Arise Between Managers and Owners)

这被称为 **代理问题 ($Agency \ Problem$)**。它源于所有权（股东）和控制权（经理）的分离。

*   **代理关系 ($Agency \ Relationship$):** 当一方（委托人, $principal$，如股东）雇用另一方（代理人, $agent$，如经理）来代表其利益时，就存在代理关系。
*   **利益冲突 ($Conflict \ of \ Interest$):** 经理的个人目标（如工作保障、更高薪水、更豪华的办公室）可能与股东最大化股票价值的目标不一致。这种冲突是代理问题的本质。
*   **代理成本 ($Agency \ Costs$):** 这些是由这种冲突产生的成本，可能是直接的（如过度的公司开支），也可能是间接的（如错失的机会）。

协调利益的机制包括基于业绩的薪酬（如股票期权）以及在经理表现不佳时面临被收购的威胁。

#### 6. 会计价值（账面价值）与市场价值的区别 (The Difference Between Accounting Value (or "Book" Value) and Market Value)

*   **账面价值 ($Book \ Value$):** 这是资产、负债和权益在资产负债表上记录的价值。它基于 **历史成本 ($historical \ cost$)**，即公司为资产支付的价格减去累计折旧。这是一个回顾性的价值。
*   **市场价值 ($Market \ Value$):** 这是资产、负债或权益今天在市场上可以实际买卖的价格。它是一个前瞻性的价值，反映了投资者对公司未来现金流和盈利能力的预期。

**对于决策而言，市场价值远比账面价值重要**，因为它反映了公司的真实经济价值。

#### 7. 会计利润与现金流的区别 (The Difference Between Accounting Income and Cash Flow)

*   **会计利润 ($Accounting \ Income$ / $Net \ Income$):** 这是利润表上的“底线”。它基于 **权责发生制 ($accrual \ principle$)** 计算，即收入在赚取时确认（不一定是在收到现金时），费用与这些收入相匹配。它还包括 **非现金项目 ($non-cash \ items$)**，例如折旧 ($depreciation$)。
*   **现金流 ($Cash \ Flow$):** 这是实际流入和流出公司的现金。它代表了可以用来偿还债务、支付股利和为新投资提供资金的“真金白银”。

关键区别在于，会计利润可能具有误导性，因为它包含非现金费用，并且可能无法反映公司的实际现金状况。**经营现金流 ($Operating \ Cash \ Flow, \ OCF$)** 是一个关键指标，用于衡量公司从正常业务活动中产生的现金。其计算公式为：
$$
OCF = EBIT + Depreciation - Taxes
$$

#### 8. 如何从财务报表确定公司的现金流——净营运资本 (How to Determine a Firm's Cash Flow from Its Financial Statements – Net Working Capital)

最重要的现金流恒等式是 **资产现金流 ($Cash \ Flow \ From \ Assets, \ CFFA$)**，它必须等于分配给投资者的现金流。它可以从两个角度计算：

**1. 资产角度:**
$$
CFFA = OCF - NCS - \Delta NWC
$$

其中:
*   $OCF \ (经营现金流) = EBIT + Depreciation - Taxes$
*   $NCS \ (净资本性支出) = 期末净固定资产 - 期初净固定资产 + Depreciation$
*   $\Delta NWC \ (净营运资本变动) = 期末 NWC - 期初 NWC$
    *   $NWC \ (净营运资本) = 流动资产 - 流动负债$

**2. 融资角度:**
$$
CFFA = 对债权人的现金流 + 对股东的现金流
$$

其中:
*   对债权人的现金流 ($Cash \ Flow \ to \ Creditors$) = 支付的利息 - 新增净借款
*   对股东的现金流 ($Cash \ Flow \ to \ Stockholders$) = 支付的股利 - 新增净股权融资

这两种方法总会得出相同的结果，验证了现金来源与去向的平衡。

### **第二部分：财务比率与规划 (Financial Ratios and Planning)**

#### 9. 如何计算和解读常用比率 (How to Compute and Interpret Common Ratios)

比率可以将财务信息标准化，对于比较和解读至关重要。

*   **流动性比率 ($Liquidity \ Ratios$ / 短期偿付能力):** 衡量公司在短期内支付账单的能力。
    *   **流动比率 ($Current \ Ratio$) = 流动资产 / 流动负债。** *解读:* 比率为1.13意味着公司每1元的流动负债，就有1.13元的流动资产作为保障。通常越高越好，但过高可能意味着资产利用效率低下。
    *   **速动比率 ($Quick \ Ratio$) = (流动资产 - 存货) / 流动负债。** *解读:* 这是一个更严格的测试，因为它剔除了流动性最差的资产——存货。比率为0.98意味着即使不出售任何存货，公司也能覆盖98%的流动负债。
    *   **现金比率 ($Cash \ Ratio$) = 现金 / 流动负债。** *解读:* 这是最保守的流动性指标，显示了可用现金可以覆盖多少短期债务。

*   **杠杆比率 ($Leverage \ Ratios$ / 长期偿付能力):** 衡量公司的长期履约能力及其财务杠杆程度。
    *   **总负债率 ($Total \ Debt \ Ratio$) = 总负债 / 总资产。** *解读:* 显示由债务融资的资产比例。比率为52.6%意味着公司52.6%的资产是通过借款融资的。
*   **周转率 ($Turnover \ Ratios$ / 资产管理效率):** 衡量公司利用其资产的效率。
    *   **存货周转率 ($Inventory \ Turnover$) = 销售成本 / 存货。** *解读:* 值为6.66意味着在一年内，公司的全部存货被销售和补充了6.66次。
    *   **存货周转天数 ($Days' \ Sales \ in \ Inventory$) = 365天 / 存货周转率。** *解读:* 这显示了存货在被售出前平均在货架上存放了多少天。值为55天意味着平均需要55天才能售出存货。

#### 10. 公司盈利能力的决定因素——杜邦恒等式 (The Determinants of a Firm's Profitability - DuPont Identity)

**杜邦恒等式 ($DuPont \ Identity$)** 是一个强大的分析工具，它将 **净资产收益率 ($Return \ on \ Equity, \ ROE$)** 分解为其三个关键组成部分，揭示了公司盈利能力的来源。

$$
ROE = Profit \ Margin \times Total \ Asset \ Turnover \times Equity \ Multiplier
$$

三个决定因素是：
1.  **利润率 ($Profit \ Margin$ / 经营效率 $operating \ efficiency$):** $净利润 / 销售额$。它衡量公司控制成本的能力。更高的利润率表明经营效率更高。
2.  **总资产周转率 ($Total \ Asset \ Turnover$ / 资产使用效率 $asset \ use \ efficiency$):** $销售额 / 总资产$。它衡量公司利用资产创造销售的效率。更高的周转率意味着更高的效率。
3.  **权益乘数 ($Equity \ Multiplier$ / 财务杠杆 $financial \ leverage$):** $总资产 / 总权益$。它衡量公司使用债务的程度。更高的乘数表明财务杠杆更大。

高 $ROE$ 可能源于高盈利能力、高效的资产使用、高杠杆或这些因素的组合。

#### 11. 如何为比较目的标准化财务报表 (How to Standardize Financial Statements for Comparison Purposes)

为了比较不同规模的公司或同一公司在不同时期的情况，我们必须对财务报表进行标准化：

*   **共同比资产负债表 ($Common-Size \ Balance \ Sheet$):** 所有科目都表示为 **总资产 ($Total \ Assets$)** 的百分比。这显示了公司的资本结构和资产组合。例如，它可以揭示现金占总资产的10%。
*   **共同比利润表 ($Common-Size \ Income \ Statement$):** 所有项目都表示为 **销售额 ($Sales$)** 的百分比。这显示了每一美元的销售收入中有多少分配给了成本、费用和利润。例如，它可以揭示销售成本占销售额的60%。

#### 12. 如何应用销售百分比法 (How to Apply the Percentage of Sales Method)

这是一种财务规划方法，用于预测未来的财务报表和外部融资需求。

**核心假设:** 大多数资产负债表和利润表科目与销售额直接相关。
**步骤:**
1.  **预测销售额:** 从一个销售预测开始（例如，增长10%）。
2.  **创建预估利润表:** 假设成本与销售额成比例变化。计算预测的留存收益增加额（$净利润 - 股利$）。
3.  **创建预估资产负债表:**
    *   将随销售额变化的资产（如现金、应收账款、存货）预测为新销售额的百分比。
    *   预测随销售额变化的自发性负债 ($spontaneous \ liabilities$)（如应付账款）。
    *   保持非自发性账户（如长期债务、普通股）不变。
    *   用步骤2中计算的增加额更新留存收益。
4.  **计算外部融资需求 ($External \ Financing \ Needed, \ EFN$):** 预测的总资产与预测的总负债及权益之间的差额就是 $EFN$。
    $$
    EFN = 预测资产 - (预测负债 + 预测权益)
    $$
    一个正的 $EFN$ 是公司必须筹集的新资金数额。

### **第三部分：货币的时间价值 (The Time Value of Money)**

#### 13. 如何确定未来收到的现金的现值 (How to Determine the Present Value of Cash to Be Received at a Future Date)

**现值 ($Present \ Value, \ PV$)** 是一笔未来资金在今天的价值，按适当的利率进行折算。这个过程称为 **折现 ($discounting$)**。

**公式:**
$$
PV = \frac{FV}{(1 + r)^n}
$$
*   $FV$ = 未来价值
*   $r$ = 每期利率
*   $n$ = 期数

这个公式将未来的现金流“带回”到其今天的等值。

#### 14. 如何确定今天投资的未来价值 (How to Determine the Future Value of an Investment Made Today)

**未来价值 ($Future \ Value, \ FV$)** 是指一项资产或现金在未来某个特定日期的价值。寻找未来价值的过程称为 **复利 ($compounding$)**，因为投资不仅在本金上赚取利息，还在已累积的利息上赚取利息。

**公式:**
$$
FV = PV \times (1 + r)^n
$$
*   $PV$ = 现值
*   $r$ = 每期利率
*   $n$ = 期数

#### 15. 利率是如何报价（和被误报）的 (How Interest Rates Are Quoted (and Misquoted))

利率主要有两种报价方式，如果不理解可能会引起混淆。

*   **年百分率 ($Annual \ Percentage \ Rate, \ APR$):** 这是法律上要求的名义利率。它的计算方法是用周期利率乘以一年中的周期数（$APR = 周期利率 \times m$）。**$APR$ 是一个“被误报”的利率，因为它忽略了一年内的复利效应。**
*   **有效年利率 ($Effective \ Annual \ Rate, \ EAR$):** 这是考虑了一年内所有复利期后的真实回报率。$EAR$ 反映了你将实际赚取或支付的利息金额。

**转换公式:**
$$
1 + EAR = (1 + \frac{APR}{m})^m
$$
*   $m$ = 每年的复利期数。

当每年复利次数超过一次（$m > 1$）时，**$EAR$ 总是会大于 $APR$**。

### **第四部分：债券与股票 (Bonds and Stocks)**

#### 16. 债券价值和收益率及其波动原因 (Bond Values and Yields and Why They Fluctuate)

*   **债券价值 ($Bond \ Value$):** 债券的价值是其所有预期未来现金流的 **现值 ($present \ value$)**。这些现金流包括定期的票息支付（一份年金, $annuity$）和最终的面值偿付（一笔一次性总付, $lump \ sum$）。
*   **债券收益率 (到期收益率, $Yield \ to \ Maturity - \ YTM$):** $YTM$ 是投资者如果持有债券至到期日可以期望获得的总回报率。它是使债券价格等于其未来现金流现值的市场利率（折现率）。
*   **波动原因:** 债券价格和收益率之间存在 **反向关系 ($inverse \ relationship$)**。
    *   当市场利率 ($YTM$) **上升**时，债券固定的未来现金流的现值会下降。因此，债券的价格会 **下跌**。
    *   当市场利率 ($YTM$) **下降**时，债券固定的未来现金流的现值会增加。因此，债券的价格会 **上涨**。
    这种波动是债券市场的一个核心特征。

#### 17. 重要的债券特征和债券类型 (Important Bond Features and Types of Bonds)

**关键特征:**
*   **面值 ($Face \ Value$ 或 $Par \ Value$):** 在期限结束时偿还的本金金额（通常为1,000美元）。
*   **票面利率 ($Coupon \ Rate$):** 债券上注明的年利率。年度票息支付额计算为 $票面利率 \times 面值$。
*   **到期日 ($Maturity \ Date$):** 发行人偿还面值的日期。

**债券类型:**
*   **附息债券 ($Coupon \ Bond$):** 在其生命周期内定期支付利息（票息）并在到期时偿还面值的标准债券。
*   **零息债券 ($Zero-Coupon \ Bond$):** 不支付任何票息的债券。它以远低于其面值的价格发行，并在到期时仅支付面值。

#### 18. 通货膨胀对利率的影响 (The Impact of Inflation on Interest Rates)

通货膨胀会侵蚀货币的购买力。为了解释这一点，我们区分两种利率：

*   **名义利率 ($Nominal \ Interest \ Rate$):** 未经通货膨胀调整的报价利率。它反映了货币 *数量* 的增长。
*   **真实利率 ($Real \ Interest \ Rate$):** 经过通货膨胀调整的利率。它反映了你 *购买力* ($purchasing \ power$) 的增长。

**费雪效应 ($Fisher \ Effect$)** 描述了这两种利率与通货膨胀之间的关系：
$$
(1 + 名义利率) = (1 + 真实利率) \times (1 + 通货膨胀率)
$$

名义利率必须足够高，以补偿投资者的真实回报率和预期的通货膨胀率。

#### 19. 利率的期限结构和债券收益率的决定因素 (The Term Structure of Interest Rates and the Determinants of Bond Yields)

*   **利率的期限结构 ($Term \ Structure \ of \ Interest \ Rates$):** 这是利率（或债券收益率）与到期时间之间的关系。它通常用 **收益率曲线 ($yield \ curve$)** 来图形化表示。
    *   **正常收益率曲线 (向上倾斜, $Upward-Sloping$):** 长期收益率高于短期收益率。这是最常见的形状。
    *   **反转收益率曲线 (向下倾斜, $Downward-Sloping$):** 短期收益率高于长期收益率。这可能是经济衰退的预兆。

*   **债券收益率的决定因素:** 债券的 $YTM$ 由加在真实利率之上的几个溢价组成：
    1.  **真实利率 ($Real \ Rate$):** 货币时间价值的基本利率。
    2.  **通胀溢价 ($Inflation \ Premium$):** 对预期未来通胀的补偿。
    3.  **利率风险溢价 ($Interest \ Rate \ Risk \ Premium$):** 对利率上升导致债券价格下跌风险的补偿。
    4.  **违约风险溢价 ($Default \ Risk \ Premium$):** 对发行人可能无法支付款项的可能性的补偿。
    5.  **税收溢价 ($Taxability \ Premium$):** 对具有不利税收待遇的债券的补偿。
    6.  **流动性溢价 ($Liquidity \ Premium$):** 对不易买卖的债券的补偿。

#### 20. 普通股 vs. 优先股（投票权） (Common Stock vs. Preferred Stock (Voting Rights))

*   **普通股 ($Common \ Stock$):** 代表公司的真正所有权。
    *   **关键权利:** 普通股股东拥有对重要公司事务的 **投票权 ($voting \ rights$)**，例如选举董事会。
    *   他们对资产和收入拥有剩余索取权 ($residual \ claim$)，意味着在清算时他们最后得到偿付。
*   **优先股 ($Preferred \ Stock$):** 一种兼具债务和权益特征的混合证券。
    *   **关键区别:** 优先股股东通常 **没有投票权**。
    *   **股利优先权:** 他们在股利支付上优先于普通股股东。
    *   **固定股利:** 优先股通常支付固定的股利，很像债券的票息支付。

#### 21. 股票估值——三种情景 (Stock Valuation – 3 Scenarios)

股票的价值是其未来预期现金流（即股利）的现值。

*   **情景1：零增长 ($Zero \ Growth$):** 股利预计将永远保持不变。股票被估值为一个永续年金 ($perpetuity$)。
    $$
    P_0 = \frac{D}{r_E}
    $$
*   **情景2：固定增长 ($Constant \ Growth$ / 戈登增长模型 $Gordon \ Growth \ Model$):** 股利预计将以一个固定的增长率 $g$ 永远增长下去。
    $$
    P_0 = \frac{D_1}{r_E - g}
    $$
    注意: $D_1 = D_0 \times (1 + g)$
*   **情景3：非固定增长 ($Non-Constant / Unusual \ Growth$):** 公司经历一段超常增长期，随后进入一个稳定的固定增长期。
    1.  预测非固定增长期的每笔股利，并计算它们的现值。
    2.  使用戈登增长模型计算固定增长期开始时的股票价格。
    3.  将这个未来的股票价格折现回今天。
    4.  将步骤1和步骤3的现值相加，得到当前的股票价格。

#### 22. 股利增长模型——资本利得收益率和股利收益率 (DGM - Capital Gain and Dividend Yield)

**股利增长模型 ($Dividend \ Growth \ Model, \ DGM$)** 可以重新排列，以显示股票的总要求回报率 ($r_E$) 由两部分组成：

$$
r_E = \frac{D_1}{P_0} + g
$$

1.  **股利收益率 ($Dividend \ Yield$):** $D_1 / P_0$。这是股票下一期的预期现金股利除以其当前价格。它代表了回报的收入部分。
2.  **资本利得收益率 ($Capital \ Gains \ Yield$):** $g$。这是股票价格预计增长的速率。它代表了回报的价格增值部分。

#### 23. 市盈率 (PE Ratio)

**市盈率 ($Price-to-Earnings \ (PE) \ Ratio$)** 是一种广泛使用的估值倍数。

*   **公式:**
$$
PE \ Ratio = \frac{每股价格}{每股收益 (EPS)}
$$
*   **在估值中的应用:** 它用于相对估值法 ($relative \ valuation$)。要为目标公司估值，你可以找到其竞争对手或行业的平均市盈率（“基准”），然后将其乘以目标公司的每股收益 ($Earnings \ per \ share, \ EPS$)。
    $$
    估计价格 = 基准 \ PE \ Ratio \times 目标公司的 \ EPS
    $$

这种方法根据市场对类似公司的估值，提供了一个快速的公司价值估算。
